What is HTML?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers receive HTML document from a web server or from local storage and render the documents into multimedia web pages. HTML describe the structure of a web page semantically and orginally included cues for the appearance of the document.

Characteristics of HTML

  1. Uses elements/tags to build web page structure
  2. Enables media files to be embedded in webpages i.e. audio and video clips
  3. Semantically defines web pages for text i.e. paragraphs, headings etc.
  4. Allows developers to embed JavaScript code
  5. Elements/tags have attributes
